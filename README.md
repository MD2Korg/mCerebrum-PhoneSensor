# Install
Download and install the latest [Study](https://github.com/MD2Korg/mCerebrum-Study/releases/latest) and [PhoneSensor](https://github.com/MD2Korg/mCerebrum-PhoneSensor/releases/latest) applications.

#### Enable unsigned applications
- Go to Settings.
- Select Security.
- On the list, Find Unknown Sources and CHECK the box.
- When done, go back to the folder.
- Tap on the apk file and select Open/Install. It should be able to install.
